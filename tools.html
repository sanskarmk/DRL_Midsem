Interactive Tools

Plug in your own numbers and watch the algorithms run, step by step. Great for checking your hand-working on practice questions.

Tools

  • Discounted-return calculator
  • Bandit Q-value tracker
  • ε-greedy exploration analyzer
  • Value-iteration replayer

1. Discounted-return calculator

Enter a reward sequence \(R_1,R_2,\dots\) and a discount \(\gamma\). Computes \(G_t=\sum_k\gamma^k R_{t+k+1}\) for every starting step.

2. Bandit Q-value tracker (single arm)

Enter the rewards observed for one arm. Choose sample-average (\(1/n\)) or constant step-size \(\alpha\). Shows the running estimate after each pull.

3. ε-greedy exploration analyzer

Enter a sequence of arm:reward pairs. Untried arms keep the initial Q (set it high for optimistic init). The tool tracks each arm’s estimate and flags every pick as DEFINITE exploration (a non-greedy arm was chosen) or possible (a greedy/tied arm — could be exploit or a lucky random pick).

4. Value-iteration replayer

Two MDPs straight from your papers. Pick γ and the number of synchronous iterations; watch the state values evolve.

The wristband run uses synchronous updates; the game run uses in-place order D, A, C, B (as the paper specifies) and fixes Goal = 100.
