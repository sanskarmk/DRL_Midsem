⭐ Straight from the professor's lecture

🎓
The exact example types flagged as “guaranteed” in class
These four mirror the numbers your professor worked through — return, state value, action value, and ε-greedy probabilities. If you can do these cold, you’ve nailed the highest-priority numerical types.
Lecture 1 · Return \(G_t\)
Rewards \(=[2,3,5]\), \(\gamma=0.9\). Find \(G_0\).
▶Show solution
\[ G_0=2+0.9(3)+0.9^2(5)=2+2.7+4.05 \]
Answer \(G_0=8.75\).
Lecture 2 · State value from returns
From state \(s\), three episodes gave returns \(10,\,8,\,12\). Estimate \(V(s)\) by Monte Carlo.
▶Show solution
\[ V(s)=\frac{10+8+12}{3}=\frac{30}{3} \]
Answer \(V(s)=10\). MC value = plain average of the observed returns from that state.
Lecture 3 · Action value from returns
State–action pair \((s,a)\) gave returns \(5,\,7,\,6\). Estimate \(Q(s,a)\).
▶Show solution
\[ Q(s,a)=\frac{5+7+6}{3}=\frac{18}{3} \]
Answer \(Q(s,a)=6\). Same averaging idea, but keyed on the \((s,a)\) pair instead of just the state.
Lecture 4 · ε-greedy probabilities
\(\varepsilon=0.2\), \(|\mathcal A|=4\) actions. Give the probability of the best action and of each other action.
▶Show solution

Best action: \(1-\varepsilon+\dfrac{\varepsilon}{|\mathcal A|}\); every other action: \(\dfrac{\varepsilon}{|\mathcal A|}\).

  1. Best \(=1-0.2+\dfrac{0.2}{4}=0.8+0.05=0.85\)
  2. Each other \(=\dfrac{0.2}{4}=0.05\)
  3. Check: \(0.85+3(0.05)=1.0\) ✓
Answer Best action \(=0.85\); each of the other three \(=0.05\).

① Returns, discounting & TD  (see Topic ①)

Drill 1.1 · Discounted return
Episodic task, rewards \(R_1{=}4,\,R_2{=}0,\,R_3{=}2,\,R_4{=}8\), discount \(\gamma=0.5\). Find \(G_0,G_1,G_2,G_3\).
▶Show solution

Work backward with \(G_t=R_{t+1}+\gamma G_{t+1}\), \(G_{\text{last}}=R_4\):

  1. \(G_3=R_4=8\)
  2. \(G_2=R_3+0.5G_3=2+0.5(8)=6\)
  3. \(G_1=R_2+0.5G_2=0+0.5(6)=3\)
  4. \(G_0=R_1+0.5G_1=4+0.5(3)=5.5\)
Answer \(G_0=5.5,\;G_1=3,\;G_2=6,\;G_3=8\). (Direct check: \(G_0=4+0.5\cdot0+0.25\cdot2+0.125\cdot8=5.5\) ✓)
Drill 1.2 · Infinite constant reward
A continuing task gives reward \(+3\) on every step forever. \(\gamma=0.9\). What is the return \(G_0\)?
▶Show solution

Geometric series \(G_0=3+3\gamma+3\gamma^2+\cdots=3\sum_{k\ge0}\gamma^k=\dfrac{3}{1-\gamma}\):

\[ G_0=\frac{3}{1-0.9}=\frac{3}{0.1}=30 \]
Answer \(G_0=30\). General rule: a constant reward \(c\) forever is worth \(c/(1-\gamma)\). This is also why \(\gamma<1\) is needed — at \(\gamma=1\) the sum diverges.
Drill 1.3 · TD(0) update
Current estimate \(V(S_t)=10\). The agent observes reward \(R_{t+1}=2\) and moves to \(S_{t+1}\) with \(V(S_{t+1})=6\). Step size \(\alpha=0.1\). Update \(V(S_t)\) for (a) \(\gamma=1\) and (b) \(\gamma=0.9\).
▶Show solution

Rule: \(V(S_t)\leftarrow V(S_t)+\alpha[\,R_{t+1}+\gamma V(S_{t+1})-V(S_t)\,]\).

  1. γ=1: target \(=2+1(6)=8\); error \(=8-10=-2\); \(V\leftarrow 10+0.1(-2)=\mathbf{9.8}\).
  2. γ=0.9: target \(=2+0.9(6)=7.4\); error \(=7.4-10=-2.6\); \(V\leftarrow 10+0.1(-2.6)=\mathbf{9.74}\).
Answer (a) 9.8   (b) 9.74. Both drop because the successor (6) is worth less than the current estimate (10) ⇒ negative TD error.
Drill 1.4 · Discount changes the verdict
Reward sequence \(R=(10,0,0,0,10)\). Compute \(G_0\) for \(\gamma=0.9\) and \(\gamma=0.5\), and say what it tells you.
▶Show solution

Only \(R_1\) and \(R_5\) are non-zero, so \(G_0=10+\gamma^4(10)\).

  1. \(\gamma=0.9:\;G_0=10+0.9^4(10)=10+0.6561(10)=10+6.561=\mathbf{16.561}\)
  2. \(\gamma=0.5:\;G_0=10+0.5^4(10)=10+0.0625(10)=10+0.625=\mathbf{10.625}\)
Interpretation With \(\gamma=0.9\) the far-off +10 still contributes 6.56 (far-sighted). With \(\gamma=0.5\) it contributes only 0.625 — a myopic agent barely values it. Same rewards, different priorities.

② Multi-armed bandits  (see Topic ②)

Drill 2.1 · Sample-average Q (stationary)
One arm returns rewards \(1,1,2,2,3\) on successive pulls. Track \(Q\) using the sample-average rule \(Q_{n+1}=Q_n+\frac1n[R_n-Q_n]\).
▶Show solution
pull nrewardupdateQ
11first sample1.000
21\(1+\tfrac12(1-1)\)1.000
32\(1+\tfrac13(2-1)\)1.333
42\(1.333+\tfrac14(2-1.333)\)1.500
53\(1.5+\tfrac15(3-1.5)\)1.800
Answer Final \(Q=1.8\) — exactly the plain mean \((1{+}1{+}2{+}2{+}3)/5=1.8\). The incremental rule just computes the running average.
Drill 2.2 · Constant-α Q (non-stationary)
Same rewards \(1,1,2,2,3\), but now use constant \(\alpha=0.3\), \(Q_1=0\). Track \(Q\).
▶Show solution

\(Q\leftarrow Q+0.3(R-Q)\):

  1. \(r=1:\;0+0.3(1-0)=0.3\)
  2. \(r=1:\;0.3+0.3(1-0.3)=0.51\)
  3. \(r=2:\;0.51+0.3(2-0.51)=0.957\)
  4. \(r=2:\;0.957+0.3(2-0.957)=1.2699\)
  5. \(r=3:\;1.2699+0.3(3-1.2699)=1.78893\)
Answer \(Q\approx1.789\). Slightly below the sample-average 1.8 here, but the key difference is that constant-α keeps weighting recent rewards more — it would track a changing arm, which the sample average cannot.

Drill 2.4 · UCB action selection
At time \(t=6\) you have arm A: \(Q=2.0,N=3\); B: \(Q=2.5,N=1\); C: \(Q=1.0,N=1\). Using UCB with \(c=2\), which arm is selected?
▶Show solution

\(\text{UCB}(a)=Q(a)+c\sqrt{\dfrac{\ln t}{N(a)}}\), with \(\ln 6=1.7918\):

  1. A: \(2.0+2\sqrt{1.7918/3}=2.0+2(0.7728)=3.546\)
  2. B: \(2.5+2\sqrt{1.7918/1}=2.5+2(1.3386)=5.177\)
  3. C: \(1.0+2\sqrt{1.7918/1}=1.0+2.677=3.677\)
Answer Select B (UCB ≈ 5.18). It wins on both counts — highest estimate and high uncertainty (tried once). Note C, despite the lowest \(Q\), outranks A purely because it is under-explored.

③ MDPs & Bellman  (see Topic ③)

Drill 3.1 · One-step Bellman (stochastic)
State \(s\), two actions, \(\pi(a_1)=\pi(a_2)=0.5\), \(\gamma=0.9\). Action \(a_1\to s_1\) (reward 5) for sure. Action \(a_2\to s_2\) (reward 10) w.p. 0.5 or \(s_3\) (reward 0) w.p. 0.5. Given \(v(s_1)=10,\,v(s_2)=20,\,v(s_3)=0\), compute \(v_\pi(s)\).
▶Show solution

Compute each action’s value, then average under \(\pi\):

  1. \(q(s,a_1)=5+0.9(10)=14\)
  2. \(q(s,a_2)=0.5[10+0.9(20)]+0.5[0+0.9(0)]=0.5(28)+0.5(0)=14\)
  3. \(v_\pi(s)=0.5(14)+0.5(14)=14\)
Answer \(v_\pi(s)=14\). (Inner sum first over outcomes \((p,r)\), then outer sum over actions weighted by \(\pi\) — exactly the state-value Bellman equation.)
Drill 3.2 · Action-value from state-values
Action \(a\) from \(s\) leads to \(s_1\) (reward 2) w.p. 0.7 and \(s_2\) (reward −1) w.p. 0.3. \(\gamma=0.8\), \(v(s_1)=5\), \(v(s_2)=10\). Find \(q_\pi(s,a)\).
▶Show solution
\[ q_\pi(s,a)=\sum_{s',r}p[r+\gamma v(s')] = 0.7[2+0.8(5)]+0.3[-1+0.8(10)] \]

\(=0.7[2+4]+0.3[-1+8]=0.7(6)+0.3(7)=4.2+2.1\).

Answer \(q_\pi(s,a)=6.3\).
Drill 3.3 · Solve \(v_\pi\) exactly (2 states) + reward scaling
A deterministic policy gives: from A → B (reward 2); from B → A (reward 4). \(\gamma=0.5\). Solve \(v(A),v(B)\) exactly. Then: what happens if all rewards are tripled?
▶Show solution

Two simultaneous Bellman equations:

\[ v(A)=2+0.5\,v(B),\qquad v(B)=4+0.5\,v(A) \]
  1. Substitute: \(v(A)=2+0.5(4+0.5\,v(A))=4+0.25\,v(A)\).
  2. \(0.75\,v(A)=4\Rightarrow v(A)=\tfrac{16}{3}\approx5.33\).
  3. \(v(B)=4+0.5(\tfrac{16}{3})=\tfrac{20}{3}\approx6.67\).
Answer \(v(A)=16/3\approx5.33,\;v(B)=20/3\approx6.67\). Tripling rewards multiplies both values by 3 (\(v(A)\to16\), \(v(B)\to20\)); the policy is unchanged because relative orderings are preserved.

Each sweep: \(V(A)=0+0.9V(B)\), \(V(B)=0+0.9V(C)\), \(V(C)=1+0.9V(T)=1\) (using the old table).

Drill 4.2 · Value iteration
States 1 and 2 (plus terminals L and R). From either state: right moves toward R, left toward L. Entering R gives +10, entering L gives 0, all else 0. Layout: L – 1 – 2 – R. \(\gamma=0.9\), \(V_0=0\), synchronous. Show 3 iterations and the greedy policy.
▶Show solution

\(V(s)=\max(\text{left}: r+0.9V(s_\text{left}),\ \text{right}: r+0.9V(s_\text{right}))\). State 1’s left = L (term, 0); state 2’s right = R (term, +10).

iterV(1) = max(left, right)V(2) = max(left, right)
1max(0, 0.9·0)=0max(0.9·0, 10)=10
2max(0, 0.9·10)=9max(0.9·0, 10)=10
3max(0, 0.9·10)=9max(0.9·9, 10)=10
Answer Converges to \(V(1)=9,\,V(2)=10\); greedy policy = go right in both states (toward the +10 goal).
Drill 4.3 · One round of policy iteration
State A has two actions: α → B (reward 20, B terminal) and β → A (reward 1, self-loop). \(\gamma=0.9\). Start from policy \(\pi=\beta\). Run policy iteration.
▶Show solution
  1. Evaluate \(\pi=\beta\): \(v(A)=1+0.9\,v(A)\Rightarrow v(A)=\tfrac{1}{0.1}=10\).
  2. Improve: \(q(A,\alpha)=20+0.9(0)=20\); \(q(A,\beta)=1+0.9(10)=10\). Since \(20>10\), switch to \(\pi=\alpha\).
  3. Evaluate \(\pi=\alpha\): \(v(A)=20+0.9(0)=20\).
  4. Improve: \(q(A,\alpha)=20\); \(q(A,\beta)=1+0.9(20)=19\). Now \(20>19\) ⇒ \(\alpha\) stays. Policy stable.
Answer Optimal policy \(\pi_*=\alpha\) (take the +20 exit), \(v_*(A)=20\). Note how the first improvement flipped β→α; the second confirmed stability.

⑤ Monte Carlo  (see Topic ⑤)

Drill 5.1 · First-visit vs every-visit
\(\gamma=1\). Episodes (state→reward→…):   Ep1: A(3) B(2) A(1) T   Ep2: B(4) A(2) T. Estimate \(V(A),V(B)\) by first-visit and every-visit MC.
▶Show solution

With \(\gamma=1\), a visit’s return = sum of rewards from that visit to T.

Ep1 returns: first A = 3+2+1 = 6; B = 2+1 = 3; second A = 1.   Ep2: B = 4+2 = 6; A = 2.

statefirst-visit returnsV (FV)every-visit returnsV (EV)
A6, 24.06, 1, 23.0
B3, 64.53, 64.5
Answer FV: \(V(A)=4.0,\,V(B)=4.5\). EV: \(V(A)=3.0,\,V(B)=4.5\). They differ for A because the second (low) A-return in Ep1 is counted by every-visit but not first-visit.

\(\gamma=1\), init \(Q=0\). Episodes (state,action,(reward)):   Ep1: S1,u,(2) S2,d,(3) T   Ep2: S1,d,(0) S2,d,(4) T. First-visit MC: find \(Q\) and the greedy policy.
▶Show solution

Returns (γ=1):

  • Ep1: \((S1,u)=2+3=5\); \((S2,d)=3\).
  • Ep2: \((S1,d)=0+4=4\); \((S2,d)=4\).

Answer Greedy policy: \(\pi(S1)=u\) (5 > 4), \(\pi(S2)=d\) (only action seen). Make it ε-greedy for continued exploration.
Drill 5.3 · Importance-sampling ratio Not in mid-sem
⛔
Out of scope for your mid-sem
Importance sampling belongs to off-policy MC, which your professor excluded from the mid-sem. Skip this drill for exam prep — it’s here for completeness.
A 3-step trajectory. Target/behaviour action probabilities along it: step 0 \(\pi{=}0.9,\,b{=}0.5\); step 1 \(\pi{=}0.2,\,b{=}0.4\); step 2 \(\pi{=}1.0,\,b{=}0.5\). Find the importance-sampling ratio \(\rho_{0:2}\).
▶Show solution
\[ \rho_{0:2}=\prod_{k=0}^{2}\frac{\pi(A_k|S_k)}{b(A_k|S_k)}=\frac{0.9}{0.5}\cdot\frac{0.2}{0.4}\cdot\frac{1.0}{0.5}=1.8\times0.5\times2.0 \]
Answer \(\rho=1.8\). The transition probabilities cancel, so only the policies enter. If any step had \(\pi=0\) (the target would never take that action), the whole product — and the weighted return — would be 0.
Drill 5.4 · Incremental MC update
A state’s current estimate is \(V=5\) after 3 returns. A 4th episode gives return \(G=8\). Update \(V\) using the incremental average \(V\leftarrow V+\frac1N(G-V)\).
▶Show solution

This is the 4th return, so \(N=4\):

\[ V\leftarrow 5+\tfrac14(8-5)=5+0.75=5.75 \]
Answer \(V=5.75\). Identical idea to the bandit/TD update — move the old estimate a \(1/N\) fraction toward the new sample.
