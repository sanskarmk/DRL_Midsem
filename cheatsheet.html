Returns & values

∑
\[ G_t=\sum_{k=0}^{\infty}\gamma^k R_{t+k+1}=R_{t+1}+\gamma G_{t+1} \]
\[ v_\pi(s)=\mathbb E_\pi[G_t\mid S_t{=}s],\quad q_\pi(s,a)=\mathbb E_\pi[G_t\mid S_t{=}s,A_t{=}a] \]
\[ v_\pi(s)=\sum_a\pi(a|s)\,q_\pi(s,a) \]
γ→0 myopic · γ→1 far-sighted. Episodic: γ=1 OK. Continuing: need γ<1.

Bellman equations

Expectation (given π)Optimality (best)
\(v_\pi(s)=\sum_a\pi(a|s)\sum_{s',r}p[r+\gamma v_\pi(s')]\)\(v_*(s)=\max_a\sum_{s',r}p[r+\gamma v_*(s')]\)
\(q_\pi(s,a)=\sum p[r+\gamma\sum_{a'}\pi q_\pi(s',a')]\)\(q_*(s,a)=\sum p[r+\gamma\max_{a'}q_*(s',a')]\)

Optimal policy: \(\pi_*(s)=\arg\max_a q_*(s,a)\).   Scaling all rewards by \(c>0\): \(v\to cv\), policy unchanged.   Adding a constant (episodic): policy unchanged; (continuing): may change.

Multi-armed bandits

🎰
\[ Q_{n+1}=Q_n+\tfrac1n[R_n-Q_n]\ \text{(stationary)};\quad Q_{n+1}=Q_n+\alpha[R_n-Q_n]\ \text{(non-stationary)} \]
Constant α = exponential recency-weighted avg \((1-\alpha)^nQ_1+\sum\alpha(1-\alpha)^{n-i}R_i\). α=1 ⇒ Q=last reward.
ε-greedy: exploit w.p. 1−ε, random w.p. ε.   UCB: \(A_t=\arg\max_a[Q_t(a)+c\sqrt{\ln t/N_t(a)}]\).
Optimistic init: high \(Q_1\) ⇒ early exploration (stationary only).
Exploratory pick? chosen arm ≠ current arg-max ⇒ definite; chosen arm = (tied) max ⇒ possible.

MAB vs MDP

Bandit = 1 state, action affects only immediate reward. MDP = action affects next state ⇒ must optimise long-run return. (Wristband Case 1 = bandit; Case 2 with battery-temp feedback = MDP.)

Dynamic programming (needs model)

🔁
Policy evaluation: \(V(s)\leftarrow\sum_a\pi\sum p[r+\gamma V(s')]\) until stable.
Policy improvement: \(\pi'(s)=\arg\max_a\sum p[r+\gamma V(s')]\).
Policy iteration: alternate eval ↔ improve until π stable (GPI).
Value iteration: \(V(s)\leftarrow\max_a\sum p[r+\gamma V(s')]\), then read off greedy π.
Synchronous: new values from old table. In-place: overwrite immediately — order matters, faster propagation.
Appropriate when: Markov + known model + small/finite state space.

Temporal difference

\[ V(S_t)\leftarrow V(S_t)+\alpha\big[R_{t+1}+\gamma V(S_{t+1})-V(S_t)\big] \]

α = learning rate (not exploration!). Bracket = TD error. Updates online, every step, model-free, bootstraps.

Monte Carlo (model-free, needs full episodes)

🎲
\(V(s)\) = average return after visits to \(s\). First-visit: first occurrence per episode (unbiased). Every-visit: all occurrences (more data, biased, both converge).
Control needs \(q\), not \(v\) (choosing from \(v\) needs the model).
Keep exploring: exploring starts · ε-soft policies · off-policy.
ε-soft: every action \(\ge\varepsilon/|\mathcal A|\). ε-greedy (special case): greedy \(=1-\varepsilon+\varepsilon/|\mathcal A|\), others \(\varepsilon/|\mathcal A|\).
Deterministic-policy MC fails: untried actions never updated ⇒ use ε-soft.

Off-policy & importance sampling Not in mid-sem

⛔
Out of scope for your mid-sem
Off-policy MC & importance sampling are not examinable (professor’s note). Listed only for completeness.
\[ \rho_{t:T-1}=\prod_{k=t}^{T-1}\frac{\pi(A_k|S_k)}{b(A_k|S_k)} \]

Model dynamics cancel (numerator = denominator) ⇒ ρ depends only on policies. Coverage: need \(b(a|s)>0\) wherever \(\pi(a|s)>0\). If the episode contains an action π never takes, ρ = 0.
